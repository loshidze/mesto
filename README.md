# Учебный проект *Mesto*

## Описание проекта
Это учебный проект  от Яндекс Практикума по курсу "Веб-разработчик". В данной работе впервые используется JavaScript.

## Используемые технологии
* Флексбоксы
* Гриды
* Адаптивная верстка с медиа-запросами
* "Попап"
* JavaScript
